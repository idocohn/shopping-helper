Hahishook Catalog

unofficial Nightly-updated static mirror of the public product catalog at hahishook.com. Built by github.com/__OWNER__/__REPO__.

Data files

  • products.json — full catalog as a JSON array
  • categories.json — every discovered category URL
  • manifest.json — {count, updated_at}

Use it from an AI assistant

Add the MCP server to your client's config (Claude Desktop, Cursor, Continue, etc.):

{
  "mcpServers": {
    "hahishook": {
      "command": "python",
      "args": ["-m", "mcp_server.server"],
      "env": {
        "HAHISHOOK_CATALOG_URL": "https://__OWNER__.github.io/__REPO__/products.json"
      }
    }
  }
}

Tools exposed: search_products, list_categories, get_product, catalog_info.

Or just curl it

curl -s https://__OWNER__.github.io/__REPO__/products.json | jq '.[0]'
