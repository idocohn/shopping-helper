Raw data files (CORS-enabled, no auth):

  • products.json — full catalog as a JSON array
  • categories.json — every discovered category URL
  • manifest.json — {count, updated_at}

An MCP server with search_products, list_categories, get_product, and catalog_info tools ships in the repo. Config snippet for Claude Desktop / Cursor / Continue:

{
  "mcpServers": {
    "hahishook": {
      "command": "python",
      "args": ["-m", "mcp_server.server"],
      "env": {
        "HAHISHOOK_CATALOG_URL": "https://__OWNER__.github.io/__REPO__/products.json"
      }
    }
  }
